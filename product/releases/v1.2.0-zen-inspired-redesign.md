## Highlights

- **Fresh color palette**: Tabby now features a refined color scheme inspired by the Zen Browser, with warmer, more cohesive tones across light and dark modes.
- **Improved visual hierarchy**: The new red accent color (replacing blue) provides better contrast and draws attention to active elements and selections.
- **Consistent styling**: All components now use a unified color system with stone/slate backgrounds and neutral foregrounds.

## Color System

### Dark Mode
- Background: Slate shades (slate-900, slate-950) for depth and contrast
- Foreground: Neutral text colors for comfortable reading
- Highlights: Light red (red-300) for selections and active states

### Light Mode
- Background: Stone shades (stone-50, stone-100) for a warm, natural feel
- Foreground: Slate text colors for clarity
- Highlights: Medium red (red-500, red-600) for selections and interactive elements

## What's Changed

### Omnibar
- Updated background from white/gray to stone/slate
- Changed selection highlight from blue to red
- Refined text colors for better readability
- Updated "Tab" type indicator from blue to red

### Tab Manager
- New stone/slate color scheme for main interface
- Red accent for selected windows and active tabs (was blue)
- Updated window selection dots with red highlights
- Refined borders and backgrounds throughout

### Options Page
- Warmer stone backgrounds instead of white
- Red primary buttons instead of blue
- Updated all text colors for consistency
- Refined card backgrounds and borders

### UI Components
- Updated tooltip backgrounds to slate
- Refreshed all input field styling
- New color scheme for empty states
- Consistent keyboard shortcut styling

## Technical Details

This update touches every visual component in Tabby, ensuring a cohesive design language throughout the extension. The new color palette was carefully chosen to:

1. Reduce eye strain with warmer tones
2. Improve focus on active elements with the red accent
3. Create better visual hierarchy across the interface
4. Maintain WCAG contrast standards for accessibility
